SkillBridge AI Pro
Software Design Document (SDD)
Version 1.0

SkillBridge AI Pro is an AI-powered Career Intelligence Platform that helps students understand their professional profile, evaluate it against job requirements, identify skill gaps, and receive personalized career guidance.


SkillBridge AI Pro - Product Requirements Document (PRD)
1. Problem Statement

Today's students face multiple problems:

They don't know whether their resume is suitable for a particular job.
ATS scores from online tools are often just keyword matchers and don't explain why the score is low.
Students don't know which skills to learn first.
Learning roadmaps are generic instead of personalized.
There is no single platform that combines resume analysis, job matching, career guidance, and AI recommendations.
Our Goal

Build an AI-powered Career Intelligence Platform that analyzes a candidate holistically and provides actionable recommendations.

2. Target Users

Primary users:

B.Tech students
Final-year students
Freshers
Internship seekers
Entry-level professionals

Future users:

Working professionals

5. Technology Stack

I want to lock this today.

Layer	Technology	Why?
Language	Python	ML ecosystem
Backend	FastAPI	Modern, fast, async
Frontend	Streamlit (V1)	Faster development; can later swap to React
Database	Supabase (PostgreSQL)	Free, authentication included
Authentication	Supabase Auth	Production-ready
PDF Parsing	PyMuPDF	Fast and reliable
NLP	spaCy	Structured text processing
Semantic Matching	Sentence Transformers	Understand meaning, not just keywords
LLM	OpenAI-compatible provider (later)	Recommendations and roadmap generation
Deployment	Render	Free backend hosting
Version Control	Git + GitHub	Industry standard

# SkillBridge AI Pro

## Project Goal

SkillBridge AI Pro is an AI-powered career intelligence platform that helps students and job seekers evaluate their resumes against job descriptions.

---

## Core Features

1. User Authentication
2. Resume Upload
3. Job Description Analysis
4. ATS Score Calculation
5. Skill Gap Detection
6. Personalized Learning Roadmap
7. Analysis History

---

## Technology Stack

### Frontend
- Streamlit

### Backend
- FastAPI

### Database
- Supabase (PostgreSQL)

### NLP
- Sentence Transformers

### Resume Parsing
- PyMuPDF
